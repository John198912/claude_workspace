# 老蒋巨靠谱 - 创作者风格分析报告

## 1. 人设与核心价值
老蒋巨靠谱，本名蒋松筠，其人设定位是一个**“接地气、共情表达”的知识型内容创作者**。他并非传统意义上的学术专家，而是以其独特的视角和亲和力，将职场、人生经验及本土化生活洞察等复杂议题，转化为大众易于理解和接受的内容。他的核心理念在于**“通过深度思考与实践，提供多元视角，帮助受众提升认知，理解社会运行法则”**。这体现在他乐于探索未知，对新兴事物抱有强烈好奇心，并坚持亲身体验和搜集一手资料，以确保内容的真实性和深度。他认为，内容创作不应止于信息传递，更应引发思考，促进个人成长。

其品牌主张可概括为**“靠谱与真诚”**。他强调内容要有人格化，通过视频露脸、使用个人头像等方式，拉近与观众的距离。在内容表达上，他反对刻意拔高立意或进行“满分小作文”式的总结，而是主张观点自然推导，让观众在观看过程中获得启发和共鸣。这种真诚的表达方式，使得他的内容更具说服力和感染力。他希望通过自己的视频，向年轻人传递丰富、多样化的内容，激发他们对社会运行法则的求知欲，并提供有价值的思考，从而缓解社会偏见，促进理性讨论。他的内容标签涵盖：**深度财经分析、职场成长指导、社会现象观察、知识科普、人文关怀与批判性思维**。这些标签共同构建了他独特的人设，使其在众多内容创作者中脱颖而出，赢得了大量受众的信任和喜爱。他的人设与核心价值，正是其能够持续吸引并留住受众的关键。

## 2. 目标人群
老蒋巨靠谱的主要受众群体是**对职场发展、个人成长、社会现象及财经知识有求知欲的年轻群体**。这部分受众通常年龄在20-40岁之间，以大学生、职场新人、中层管理者以及对社会议题有独立思考能力的青年为主。他们普遍面临着职场晋升的压力、个人成长的困惑、社会变迁带来的焦虑，以及对知识付费、消费主义等现象的辨析需求。

他们的**痛点**主要包括：
1.  **职场迷茫与焦虑：** 对职业发展路径不清晰，面对职场不公或挑战时感到无助，渴望获得实用的职场经验和应对策略。
2.  **个人成长瓶颈：** 渴望提升认知层次、拓宽思维边界，但缺乏有效的方法和指导，容易陷入信息茧房或被碎片化知识误导。
3.  **社会现象困惑：** 对社会热点、经济趋势、文化现象等抱有疑问，希望获得深度解读和多元视角，以形成自己的独立判断。
4.  **信息过载与辨别：** 面对海量信息，难以辨别真伪，渴望有权威且接地气的声音进行引导和分析。

他们的**需求**则体现在：
1.  **实用性知识：** 渴望获得能够指导职场实践、提升个人能力的具体方法和建议。
2.  **深度思考与启发：** 不满足于表层信息，希望通过内容获得对事物本质的理解，激发批判性思维。
3.  **情感共鸣与认同：** 希望内容创作者能够理解他们的困境和感受，提供情感上的支持和共鸣。
4.  **多元化视角：** 渴望接触不同于主流的观点，拓宽视野，避免思维定势。

老蒋巨靠谱通过其“接地气、共情表达”的风格，以及对职场、人生经验和本土化生活洞察的专注，精准地触达了这部分受众的痛点和需求，为他们提供了有价值的解决方案和思考框架。

## 3. 思维模式与人格高度
老蒋巨靠谱的思维模式展现出**深度分析与批判性思考**的特质。他本科学习IT，后转为记者，这种跨学科背景使其在面对问题时，既能保持理工科的严谨逻辑，又能兼具人文社科的广阔视野。他善于从现象中挖掘本质，不满足于表层信息，而是通过大量调研和亲身体验，获取一手资料，进而构建起对复杂议题的深刻理解。

他的世界观是**开放且多元的**，不拘泥于单一视角，力求在内容中呈现多方观点，引导受众进行独立思考。他曾表示希望自己的视频能够最大限度地做到角度的差异化，从而传递多元的观点，缓解社会偏见。这表明他具备较高的认知层次，能够超越个体经验，从更宏观、更系统的层面审视问题。

在认知层次上，老蒋巨靠谱展现出从**信息整合到知识体系构建，再到价值观输出**的递进。他不仅是信息的搬运工，更是知识的提炼者和思想的启发者。他通过思维导图等方式辅助观众理清复杂逻辑，体现了其将复杂知识结构化的能力。他的价值观输出并非生硬灌输，而是通过对社会现象的剖析、职场经验的分享，以及对人生哲理的探讨，自然而然地融入内容之中，引导观众形成积极、理性的价值观。

他的人格高度体现在其**真诚、负责任**的态度。他曾强调涨粉的关键是真诚，不刻意迎合，而是坚持输出有价值、有深度的内容。在面对争议话题时，他敢于表达自己的独立见解，并尝试通过对话和分析来还原真实情况，缓解误解。这种独立思考和勇于担当的精神，使其在受众心中树立了“靠谱”的形象。

## 4. 内容风格与结构
老蒋巨靠谱的内容风格以**深度、理性、接地气**为核心特征，其内容形式主要以**中长视频**为主，辅以**播客**等形式。在视频中，他常以本人出镜的方式，佩戴黑框眼镜，穿着卫衣，营造出一种亲和、理工男式的形象，实现了内容的**人格化表达**。这种风格打破了传统财经内容的严肃刻板，使其更具吸引力。

在**叙事结构**上，老蒋巨靠谱的内容通常遵循**“问题提出—背景分析—深度调研—多方观点呈现—逻辑推导—价值输出”**的模式。他擅长将复杂、抽象的财经或社会议题，通过**科普化**的方式进行拆解，并辅以**思维导图**等视觉工具，帮助观众理清逻辑脉络。例如，在分析知识付费、饭圈经济等话题时，他会从现象入手，深入挖掘其背后的商业逻辑、社会心理和文化成因。他曾提到，视频文案的写作逻辑与文字报道不同，需要**“每一两分钟安排一个兴奋点，或者让网友有获得感”**，这表明其在内容节奏和信息密度上进行了精心设计，以适应视频这种线性传播媒介的特点。

**表达风格**方面，老蒋巨靠谱以**共情、真诚且略带批判性**的语言特色著称。他善于从普通人的视角切入，引发观众共鸣，同时又不失独立思考和批判精神。他反对刻意拔高立意或进行“满分小作文”式的总结，而是主张**观点自然推导**，让观众在观看过程中自行领悟。这种不预设结论、引导思考的表达方式，增强了内容的说服力和深度。他的语言平实、口语化，避免使用过于专业的术语，即使涉及复杂概念也能用通俗易懂的例子进行解释，使得内容更具传播性。

此外，老蒋巨靠谱的内容还体现出**对一手资料的重视和亲身体验**。他为了制作视频，会亲自下载体验近20个社交软件，或连续多日观察直播带货，甚至与粉丝连麦对话，这种严谨的调研态度，确保了内容的真实性和权威性。这种**“从实践中来，到实践中去”**的创作理念，使其内容不仅有理论深度，更有现实温度。

## 5. 爆款内容公式
老蒋巨靠谱的爆款内容公式，虽然没有明确的“标准化内容生产模式”被公开，但从其内容特点和访谈中可提炼出以下要素：

**钩子设计：**
*   **热点议题切入：** 紧跟社会热点和大众关注的痛点问题，如知识付费、职场困境、社会舆论、AIGC发展等，引发观众共鸣和好奇心。例如，早期的《每天看罗振宇能自我提升吗？知识付费是不是智商税》以及近期的《影视飓风的舆情，说明互联网已经烂完了》都属于此类。
*   **反常识或颠覆性观点：** 提出与大众普遍认知不同，但经过深度思考和调研支撑的观点，激发观众的求知欲和讨论欲。例如，对知识付费的批判性分析。
*   **个人经验与故事：** 结合自身或他人的真实经历，增加内容的代入感和可信度，让观众更容易产生共情。

**核心框架：**
*   **问题导向型：** 从一个具体问题或现象出发，层层深入，剖析其本质和深层原因。
*   **多维度分析：** 运用财经、社会学、心理学等多学科视角，对问题进行全面解读，提供丰富的信息增量。
*   **逻辑严谨，论证充分：** 强调深度调研和一手资料的重要性，通过数据、案例、访谈等方式支撑论点，增强内容的权威性和说服力。他曾提到“没有一套可重复的方法论，都是调研后进行归纳总结”，这表明其内容框架是基于扎实研究而非套路。
*   **思维导图辅助：** 在视频中常使用思维导图，帮助观众梳理复杂逻辑，降低理解门槛。

**情感曲线：**
*   **共情开场：** 通过对观众痛点的精准把握，迅速建立情感连接，引发共鸣。
*   **理性分析与思考：** 在内容主体部分，保持冷静客观的分析态度，引导观众进行理性思考，而非煽动情绪。
*   **启发与反思：** 结尾部分通常不给出标准答案，而是提供多元视角，鼓励观众独立思考，引发对自身和社会的深层反思。他曾表示“不刻意立意，自然而然地推导出观点比较好”，避免了说教感。

**标准化内容生产模式：**
虽然老蒋巨靠谱表示“没有一套可重复的方法论”，但其创作流程仍具有一定的标准化特征：
1.  **选题确定：** 结合社会热点、个人兴趣和受众需求，确定具有讨论价值的议题（约2天）。
2.  **深度调研：** 投入大量时间进行资料搜集、亲身体验、专家访谈等，获取一手信息（3-7天）。
3.  **内容逻辑梳理与文案撰写：** 将调研结果进行归纳总结，构建内容框架，撰写视频文案，注重兴奋点和获得感。
4.  **视频制作与剪辑：** 拍摄、剪辑，并运用思维导图等辅助工具，确保内容呈现的清晰度和吸引力（剪辑耗时较长，如50小时）。
5.  **多平台分发与互动：** 在B站、YouTube、抖音、微信视频号等平台发布，并关注用户反馈。

## 6. 核心内容框架
1.  **“问题导向+深度剖析”框架：** 老蒋巨靠谱的内容创作常以社会热点、职场痛点或认知误区等具体“问题”为切入点。他不会直接给出结论，而是引导观众深入思考问题背后的成因、逻辑和多方影响。此框架强调通过**背景分析、多维度调研、案例支撑**等方式，对问题进行层层剥茧的深度剖析，最终呈现一个全面、立体的认知图景。例如，对知识付费、饭圈经济等话题的探讨，均遵循此模式，旨在帮助受众理解复杂现象的本质，而非停留在表面。

2.  **“科普化+人格化”表达框架：** 针对复杂或专业的财经、社会议题，老蒋巨靠谱采用**科普化**的语言和叙事方式，将晦涩难懂的概念转化为通俗易懂的表达。同时，他通过本人出镜、亲和的形象以及真诚的表达，实现内容的**人格化**。这种框架使得内容既有知识的深度，又不失温度和亲近感，有效降低了受众的理解门槛，增强了内容的吸引力和传播力，让观众在轻松的氛围中获取知识和启发。

3.  **“实践调研+批判性反思”框架：** 老蒋巨靠谱非常重视**一手资料的获取和亲身体验**。他会投入大量时间进行实地调研、访谈或亲身参与，以确保内容的真实性和权威性。在此基础上，他会结合自身的独立思考，对现象进行**批判性反思**，不盲从主流观点，而是提出独到见解。此框架强调从实践中发现问题，通过严谨调研获取数据，再结合批判性思维进行分析，最终形成有深度、有说服力的内容。

4.  **“多元视角+价值引导”框架：** 在内容呈现上，老蒋巨靠谱力求展现**多元化的视角**，避免单一和片面的观点。他会主动收集和呈现不同立场、不同背景的观点，鼓励观众进行独立思考和判断。在此过程中，他并非简单罗列观点，而是通过其独特的思维框架和价值观，对这些多元视角进行梳理和引导，最终传递出积极、理性、有建设性的价值观，帮助受众形成健全的世界观和认知体系。

## 7. 平台策略
老蒋巨靠谱在**B站、YouTube、抖音和微信视频号**等多个平台活跃，其平台策略体现出对不同平台特性和算法机制的适配。

1.  **B站（Bilibili）：** 作为其主要阵地，老蒋巨靠谱在B站以**中长视频**为主，深耕知识区和财经区。他曾是B站较早一批做严肃财经内容的UP主，其内容深度和价值吸引了大量年轻用户。在B站，他注重内容的**科普化和人格化**，通过露脸、思维导图等形式，降低财经知识的理解门槛。他关注B站的内容生态，对平台“下沉内容”的崛起有自己的看法，并积极参与到与算法相关的讨论中。B站的算法更倾向于**内容质量和用户粘性**，老蒋巨靠谱通过高质量、有深度的内容，培养了忠实的用户群体，并通过互动保持活跃。

2.  **YouTube：** 作为国际平台，YouTube上的内容与B站有一定重合，但可能在选题上更具国际视野或更偏向普适性话题。YouTube的算法也注重**观看时长和用户互动**，老蒋巨靠谱的长视频内容符合这一特点。

3.  **抖音（Douyin）和微信视频号：** 在这些短视频平台，老蒋巨靠谱可能采取**内容切片或精华提炼**的方式，将长视频中的核心观点或精彩片段进行二次创作，以适应短视频**快节奏、碎片化**的消费习惯。抖音的算法更侧重于**完播率、互动率（点赞、评论、转发）及复播率**，因此在这些平台，内容开头（前3秒）的钩子设计至关重要。微信视频号则更依赖**社交推荐**，通过微信生态的分享机制进行传播。

总体而言，老蒋巨靠谱的平台策略是**“一源多用，因地制宜”**。他以深度内容为核心，根据不同平台的受众特点和算法偏好，调整内容的呈现形式和传播策略，以实现内容价值的最大化和影响力的广泛覆盖。
