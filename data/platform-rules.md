# 平台规则轻量层 · platform-rules

> 版本：V2.2（2026-06-12）| 用途：为 M4 适配/发布与 `publish_package_gate` 提供单一、可核验的平台硬限制来源。
> 设计取舍：**轻量优先**。只记录会导致内容被拒/限流/违规的硬限制 + 核验来源 + 核验日期。**不做** `change_log` 版本机、不做强制 schema。
> 维护规则：每条规则带 `last_verified`；超过 90 天未核验，`publish_package_gate` 应将其标为 `medium` 提醒（非阻断）。
>
> 核验方法（2026-06-12）：Web 搜索交叉验证 + 官方页面确认。source_url 指向官方页面。

---

## 字段说明

| 字段 | 含义 |
|------|------|
| `platform` | 平台标识 |
| `hard_limits` | 硬限制（超出会被截断/拒绝/限流） |
| `forbidden_terms` | 平台敏感/导流违规词 |
| `soft_recommendations` | 软建议（不阻断，仅提醒） |
| `source_url` | 规则来源链接（官方优先） |
| `last_verified` | 最近一次人工核验日期 |

## 小红书 (xiaohongshu)

- **hard_limits**：
  - 标题 ≤ 20 字（超出被截断）；建议前 12 字点明核心价值
  - 图片 ≤ **9** 张（⚠️ 旧版写 18 张，已核验修正为 9 张）
  - 话题标签 ≤ **10** 个（⚠️ 旧版写 20 个，已核验修正）
  - 视频单次最多 1 个，≤ 4GB
  - AI 生成内容须标注「AI 辅助创作」
  - 2026 年 3 月起新号无新手保护期，从首条笔记起严格审核
- **forbidden_terms**：站外导流词（微信/公众号/加我/二维码）；绝对化用词（最/第一/100%）；医疗功效断言；硬广直接下架；关键词堆砌；虚假人设；低俗擦边
- **soft_recommendations**：封面首选 3:4 竖版；正文 ≥ 100 字，长笔记可超 600 字；emoji 适量；效果对比图须标注「原相机 | 日期」
- **source_url**：
  - 小红书社区公约 App 内路径：「我的」→ 左上角菜单 → 小红书社区公约
  - 社区商业公约（官方）：https://school.xiaohongshu.com/rule/detail/5d67e65f0000000000000000/62738ce116a4f800136cbe3a
  - 《社区公约 2.0》发布公告（2026-01-19）：https://mp.weixin.qq.com/s?__biz=Mzg4Mzg0NDQ2Ng==&mid=2247700361
- **last_verified**：2026-06-12（✅ 已核验）

## 公众号 (wechat)

- **hard_limits**：
  - 封面标题 ≤ **64 字节**（即 32 个汉字）；超出不显示
  - 订阅号每天群发 **1 次**，每次最多 **8 篇**
  - 服务号每月 4 次
  - AI 生成内容须在正文首段显著标注，否则取消原创、限流、封号
  - 文字重复率 > 15% 触发审核；结构/观点相似度 > 60% 认定为洗稿，三次永久封号
- **forbidden_terms**：诱导分享/诱导关注话术；夸大医疗/金融收益；硬广互推（不进入推荐池）；营销词占比 > 15% 驳回；未标注「广告」的商业推广
- **soft_recommendations**：每段 ≤ 5 行；重点加粗；开头 30 字告知价值；完读率是核心流量指标（2026 年起）
- **format_compat**（编辑器硬兼容事实，排版时必须遵守；排版决策心法见 `skills/m4-distribution/s-4.2.2-adapt.md` 公众号段）：
  - 仅支持**纯内联样式**：CSS 必须写在元素 `style` 属性里，外部 `<style>` 标签与 class 会被剥除
  - `<blockquote>` 样式失效：2024-11 起微信编辑器剥离 blockquote 的边距/竖线样式，引用块改用 `<section style=...>` 实现
  - `<ul>/<ol>` 列表样式不稳定：项目符号/缩进常被吞，建议用 `<section>` + flexbox 模拟列表
  - `margin` 简写被丢弃：`margin: 0 0 1em` 这类分拆写法会被清掉，须合并为单值或用 `margin-top/margin-bottom` 分别写
  - **外链失效**：正文超链接（非公众号内部）不可点击，须转为文末脚注/参考列表
  - **不支持 Mermaid / JavaScript / SVG 动态素材**：流程图须转为静态 PNG 图片后插入
  - 中文正文用半角标点（`, : ; ? ! . ( )`）是典型 AI 味且不符合中文排版，须转全角（`，：；？！。（）`）
- **source_url**：
  - 微信公众平台运营规范（官方）：https://mp.weixin.qq.com/mp/opshowpage?action=newoplaw
  - 微信公众平台规则中心：公众号后台 → 设置 → 规则中心
- **last_verified**：2026-06-12（✅ 已核验）

## 知乎 (zhihu)

- **hard_limits**：
  - 文章标题 ≤ **30 字**（超出截断显示）；建议 5-30 字
  - 提问标题 1-50 字，不得含斜杠、特殊符号、广告词、人身攻击
  - 想法 ≤ 500 字；评论 ≤ 2000 字
  - AI 生成内容须标注「本内容由 AI 辅助生成」，否则降权或折叠
- **forbidden_terms**：站外导流（二维码/联系方式）；营销硬广未注明利益相关；洗稿抄袭；重复刷屏；标题党；冒充医生/律师等专业身份；地域/性别/职业歧视
- **soft_recommendations**：回答建议 500-3000 字，低于 200 字的回答较难获得推荐；开头先给结论；增加数据来源标注
- **source_url**：
  - 知乎创作者中心 → 规则与协议
  - 知乎社区规范（App 内可查阅）：https://www.zhihu.com/
- **last_verified**：2026-06-12（✅ 已核验）

## 抖音 (douyin)

- **hard_limits**：
  - 视频标题约 **30-40 字**（超出在推荐流中被截断）
  - 视频简介约 **1000 字**上限
  - 封面禁用「热播」「爆款」「强推」等营销字样
  - AI 生成占比 ≥ 50% 的作品，须在封面和简介两处标注「本作品含 AI 生成内容」
  - 评论单条 ≤ 100 字
- **forbidden_terms**：违规导流；绝对化用词；未标注的商业推广；封面标题擦边但内容违规；医疗/金融虚假宣传
- **soft_recommendations**：前 3 秒留人；结尾 CTA（行动号召）；完播率和互动率是核心流量指标
- **source_url**：
  - 抖音社区自律公约（官方）：https://www.douyin.com/rule/policy
  - 抖音 App 内路径：「我」→ 右上角 ☰ → 社区评审团
- **last_verified**：2026-06-12（✅ 已核验）

## B站 (bilibili)

- **hard_limits**：
  - 无官方硬性字数规定（显示限制：手机端 ≈ 24 字、PC 端 ≈ 22 字）
  - 封面与口播须符合社区规范
  - 禁止色情低俗、暴力、侵权、虚假宣传
- **forbidden_terms**：侵权内容；色情低俗；刷量造假；欺诈信息；冒充他人；人身攻击；未成年人安全违规
- **soft_recommendations**：标题建议 ≤ 20 字；排行榜标题建议 10-16 字；封面清晰有吸引力；前 3 秒留人
- **source_url**：
  - B站社区规范（英文版，2025-03-03 更新）：https://www.bilibili.tv/marketing/protocal/communityrules_en.html
  - B站创作者中心 → 社区规范
- **last_verified**：2026-06-12（✅ 已核验）
