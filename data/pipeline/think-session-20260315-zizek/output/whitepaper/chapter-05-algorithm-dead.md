# 第五章：算法已死——大他者、企业封建主义与平台经济的神学批判

> 白皮书：否定性的实践学 | 章节 5/8

## 看不见的手其实是隐身的独裁者

亚当·斯密说：个体追求私利，通过一只"看不见的手"，会自动产出公共福祉。弗里德里希·哈耶克将其升级为"自发秩序"（spontaneous order）：不需要中央计划，市场价格机制本身就能高效配置资源，任何试图人为干预的行为都不如"让市场自己来"。

这听起来像是在描述一个没有主人的系统。但齐泽克看穿了它的本质：

**哈耶克的市场不是"没有主宰"——它是把主宰去主体化了。**

"看不见的手"不是真的不存在。它只是让自己变得不可见、不可捉摸、不可问责。市场变成了一个全知全能的、超越人类理解能力的实体——你必须像信徒一样屈从于它，承认它的智慧远在你之上。

齐泽克指出：这就是拉康说的**"大他者"（The Big Other）**的终极形态——一个没有面孔、没有地址、没有意图，但却主宰一切的隐形权威。信徒不需要见过上帝，只需要相信"在某处有一个知道一切的实体在安排着"。市场/算法扮演了同样的角色。

## 算法：数字时代的大他者

现在把这个逻辑映射到2025年的平台经济。

抖音有算法。小红书有算法。微信公众号有算法。YouTube有算法。每一个创作者都知道——你的内容能不能被看到，不取决于内容本身的质量，而取决于**算法是否"选中"了你**。

创作者对算法的态度，精确地复制了信徒对上帝的态度：
- **不可知**：没有人真正知道算法的完整规则。平台自己也不完全公开。
- **不可抗**：你可以猜测、可以迎合、可以"优化"，但最终决定权不在你。
- **不可怨**：当你的内容没有流量时，你不会去质疑算法本身是否合理——你会反思自己是否"做得不够好"。
- **需要献祭**：为了被算法"选中"，你必须调整你的标题、封面、关键词、发布时间——甚至你的内容本身。

这是市场神学的数字版本。算法就是2025年的"看不见的手"——去主体化的、全知全能的、不可问责的大他者。

## 企业封建主义

齐泽克更进一步，提出"企业封建主义"（corporate feudalism）的诊断：在数字平台经济中，科技巨头不是传统意义上的"雇主"，而是**收取地租的数字领主**。

封建制度的核心不是"某个坏国王在压迫你"，而是一种**产权结构**：领主拥有土地，农奴在土地上劳作，产出的大部分以"地租"的形式上缴给领主。农奴"自由地"在土地上耕种，但这个"自由"是虚假的——因为土地不是他的。

平台经济的结构几乎完美重演：
- **平台**=土地所有者（数字地主）
- **创作者**=在平台上劳作的数字农奴
- **内容流量**=收成
- **平台抽成/算法控制**=数字地租

你"自由地"在抖音上创作内容。但抖音拥有分发权（土地）、拥有你的受众关系（农奴不能把土地带走）、拥有规则制定权（领主的司法权）。你的"自由"是在别人的土地上耕作的自由。

而凌晨一点焦躁地修改标题关键词的那个博主，正在执行一套精密的**献祭仪式**——就像中世纪的农奴在领主的庄园前恭敬地呈上贡品。

## 两种"自发性"：精确的本体论坐标

在我们的对话中，出现了一个关键问题：如果齐泽克反对哈耶克的"自发秩序"，那他自己的替代方案难道不也依赖某种"自发性"吗？

答案是：齐泽克的"自发性"与哈耶克的"自发性"存在一个精确的本体论差异，坐标如下：

| | 哈耶克 | 齐泽克 |
|---|--------|--------|
| **自发的主语** | 系统（自发演化） | 主体（自发决断） |
| **大他者状态** | 隐秘存在且全能 | 已死且不复存在 |
| **主体姿态** | 逃避自由，交出决断权 | 直面深渊，承担责任 |
| **"自然"的含义** | 秩序如植物般自然生长 | 暴力打断"自然化"幻觉 |

哈耶克的"自发"意味着："什么都不用管，系统会自动安排好一切。"——大他者在运作。

齐泽克的"自发"意味着："根本没有系统或神明来为我们担保，除了我们彼此之间激进的政治结盟，一切空无一物。"——大他者已死。

## "算法已死"：一个创作者的本体论宣告

如果我们认真对待齐泽克的诊断，那么对创作者来说，有一个无法回避的结论：

**真正的超级个体不是一个讨好算法的熟练奴仆，而是一个宣告"算法已死"的召集人。**

这不是呼吁你删掉所有社交媒体账号（那是另一种逃避）。而是一个本体论层面的姿态转换：从"我必须搞懂算法的规则然后迎合它"（信徒朝拜大他者），到"算法不过是一套代码，它不能为我的创作赋予意义，也不能替我做关于'为谁写作'的决定"（宣告大他者之死）。

那么，在大他者死后的废墟上，创作者和受众之间的关系应该被如何重建？
