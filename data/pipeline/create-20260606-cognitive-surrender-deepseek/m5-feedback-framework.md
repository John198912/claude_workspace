## 四、策略优化触发规则

### 触发条件（OR逻辑）

| 条件 | 触发动作 |
|------|---------|
| 公众号分享率 > 5% | → 追加一篇"认知投降"姊妹篇（如"借来的信心"选题卡V3-05） |
| B站完播率 > 50% | → 追加B站"认知投降"系列（如GPS类比篇/10分钟篇/台北篇 分别深化） |
| 小红书收藏率 > 10% | → 追加小红书"反投降实操清单"图文系列 |
| 抖音评论集中在"怎么办" | → 追加抖音行动指南短视频 |
| 任何平台出现"看不懂"评论 > 5条 | → 追加一篇"认知投降·小白入门版"（降低概念门槛） |
| "认知投降"被至少3个其他创作者引用 | → 追加品牌定位文章："为什么我们率先提出认知投降" |
| 72h内任一平台KPI低于基准线50% | → 触发内容诊断（s-3.0-content-diagnostics） |
| 30天收藏量持续增长 | → 该文章升级为SOUL"常青内容"，纳入新人引导流程 |

### 迭代方向预案

| 受众反馈类型 | 内容迭代方向 |
|-------------|------------|
| "太学术/太复杂" | 降低概念密度，增加故事比例，拆分系列 |
| "不够深" | 追加单篇深度解析（如只讲Shaw&Nave实验设计） |
| "和我的工作/学习怎么结合？" | 追加场景化操作指南（程序员版/学生版/管理者版） |
| "是不是在贩卖焦虑？" | 追加正面叙事——"AI时代认知主权的重建" |
| "有没有更多同类研究？" | 追加学术综述——"认知投降研究全景地图" |

### 因果纪律（依据 QUALITY-GATES.md）

所有M5结论必须标注因果层级：
- `descriptive`：描述性事实（如"公众号分享率高于B站投币率"）
- `correlational`：相关关系（如"收藏率高与自测清单形式有正相关"）
- `causal`：因果关系——**仅在A/B实验或准实验设计下使用**（当前框架无A/B，不使用causal）
