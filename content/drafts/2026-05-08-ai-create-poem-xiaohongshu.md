## 图3：AI的工作原理 - 万花筒

### 视觉设计

**主视觉**：
- 中心：万花筒动画示意图
- 周围：3个步骤的流程图

**标题**：
```
🎨 AI = 万花筒
```

**内容**：
```
万花筒里有无数彩色碎片
每转一次，碎片重新组合
产生"新"图案

但所有图案都"一样"
因为都是那些碎片的重组

━━━━━━━━━━━━━━━

AI写诗的3步：

1️⃣ 识别关键词
   输入："写一首关于孤独的诗"
   AI识别：[孤独] [诗] [情感]

2️⃣ 检索相关元素
   月光（相关性：0.89）
   空荡（相关性：0.87）
   影子（相关性：0.85）

3️⃣ 重组元素
   从10亿种组合中
   找到统计最优解

━━━━━━━━━━━━━━━

所以AI说"我创作了"
其实是说：
"我从10亿种组合中
找到了统计上最优的那一个"

这不是创作
这是...重组
```

**底部强调**：
```
💡 AI的动词：recombine（重组）
```

## 图4：人类的创作 - 雕刻刀

### 视觉设计

**主视觉**：
- 中心：雕刻刀和石头
- 米开朗基罗的大卫像剪影

**标题**：
```
🔨 人类 = 雕刻刀
```

**内容**：
```
米开朗基罗说：
"雕塑已经在石头里了
我只是把多余的部分去掉"

━━━━━━━━━━━━━━━

一个诗人写诗的过程：

第一稿：
"我一个人坐在房间里，很孤独"
→ 划掉（太直白）

第二稿：
"月光洒在空荡的房间，我感到孤独"
→ 划掉（太常见）

第三稿：
"我妈妈去世后
我学会了一个人吃饭"
→ 保留（这就是我要说的）

━━━━━━━━━━━━━━━

注意这个过程：

诗人不是在"添加"元素
而是在"删除"可能性

每一次删除，都是一次选择
每一次选择，都是一次放弃
```

**底部强调**：
```
💡 人类的动词：select（选择）
```

```
语言不是用来"描述世界"的
```

```
一个自媒体人想写关于失去母亲的文章

━━━━━━━━━━━━━━━

第1步：让AI生成5个开头
→ 你选择："我妈妈去世后..."

第2步：让AI扩展这个方向

→ 你选择：克制版的结构

第4步：你决定改什么、留什么
→ 你想起：妈妈最爱做番茄炒蛋

━━━━━━━━━━━━━━━

💡 什么是"选择链"？

就是你做出的所有选择：
• 选择哪个方向（5选1）
• 选择哪个版本（3选1）
• 选择删除什么（删除80%）
• 选择添加什么（番茄炒蛋）

选择链越长，作品越"你"
```
