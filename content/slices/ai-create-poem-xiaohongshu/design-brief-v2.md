
```

字体：思源黑体 Bold
主标题：52px
```


#### 设计说明

**背景**：米白色渐变（上 #FEF9F3 → 下 #FFF5E6）

**左侧区域 - 电脑屏幕**：
- 位置：画面左下方，占约40%宽度
- 外形：圆角矩形（笔记本电脑或显示器轮廓），浅蓝色发光边框
- 内容：排列整齐的文字行，模拟"10首诗标题列表"
- 风格：完美、整齐、冷蓝光
- 实现：用灰色细线模拟文字行即可

**右侧区域 - 手写纸张**：
- 位置：画面右下方，占约40%宽度
- 外形：略微倾斜的纸张，有折痕/褶皱纹理
- 内容：大量划线删除的文字，只有最底部1行被红笔圈出
- 被圈文字用红色圆角矩形框标注
- 风格：粗糙、真实、有人情味

**中间 - 人物剪影**：
- 位置：画面上方中央，连接左右两个区域
- 形状：侧坐的人，一只手托腮（思考状）
- 颜色：深棕色半透明剪影 (#3D2817, opacity 0.6)

**文字位置**：
- 主标题"AI写了10首诗"：右上区域，52px Bold，深棕色
- 主标题"我只留下1句话"：紧接上一行
- 重点数字"10"和"1"：用橙色(#FF6B35)强调
- 副标题"这才是真正的创作"：主标题下方，44px Regular，深棕色
- 标签行：底部居中，28px，浅灰色(#9E9E9E)
