# B站视频制作设计方案

## 一、视频总览

| 维度 | 规格 |
|------|------|
| 片长 | 23分钟（20-25分钟区间内） |
| 分辨率 | 1080p 60fps（或 4K 上传） |
| 画幅 | 16:9 横版 |
| 章节数 | 7章（0-7） |
| 出镜 | 创作者正脸出镜（画中画 + 正片穿插） |
| 风格 | 深度分析 + 实机演示 + 动画辅助 |
| 方法论 | 三阶对话法（螺旋结构） |

---

## 二、叙事结构（三阶对话法应用）

```
第一层·场景爆破（0:00-3:30）
  → 开场Hook + 思想实验 → 打破"AI在创作"的默认假设

第二层·结构拆解（3:30-15:00）
  → AI工作原理 + 人类创作过程 + 命名时刻 + 哲学支撑
  → 给出新框架：recombine vs select / 万花筒 vs 雕刻刀

第三层·反刍重建（15:00-21:30）
  → 实机演示 + 选择链概念 + 回到开场
  → 给可用的思维工具

螺旋回归（21:30-23:00）
  → 金句收尾 → 下期预告 → CTA
  → 回到原点，站位更高
```

## 十、标题/封面/简介

### 推荐标题

**主标题**："AI能写1000首诗，但这不是创作。"

**副标题**："recombine vs select——第二课：关于人机协作"

### 封面设计

- **左右冷暖分屏**：左-蓝色冷调万花筒图案，可见重复的文字碎片。右-暖橙色手握住雕刻刀，正在削减一块"文字石"
- **中间大字**："AI写诗 vs 你写诗"
- **底部小字**："这就是重组与选择的区别"
- **调性**：深蓝(#1a237e) + 暖橙(#FF6B35) + 白色文字
- **避免**：黄底黑字（太营销号）、小字堆积

### 视频简介

```
【AI能写1000首诗，但这不是创作。】

让AI写10首关于"孤独"的诗，每首都"不错"——月光、空荡、影子、寂静——但每首都"差不多"。让10个人写"孤独"——每首都完全不同。

为什么？

答案不在于"AI会不会创作"，而在于我们一直在用错误的动词。

AI在重组（recombine），你在选择（select）。AI是万花筒，你是雕刻刀。

这一期，我用一个思想实验+实机演示，聊聊AI创作的真相。

【时间轴】
0:00 深夜11点，一个设计师的困惑
0:45 思想实验：10首AI诗 vs 10个人的诗
3:30 万花筒：AI的工作方式
7:00 雕刻刀：人类是怎么创作的
11:00 命名时刻：recombine vs select
12:30 维特根斯坦说了什么
15:00 实机演示：5步选择链
19:30 你的选择链有多长？
21:30 核心金句+下期预告

【相关视频】
→ 上期：当AI说"我理解你"，它在说什么？
→ 下期预告：如果AI能做一切，那我该做什么？

【评论区聊聊】
你的"选择链"有多长？0次？10次？100次？
```

### 置顶评论

```
【关于这期视频的几个补充】

1. 维特根斯坦的"语言游戏"是很深的概念，这期只讲冰山一角。想深入了解的朋友，推荐《哲学研究》（不是《逻辑哲学论》）。

2. 实机演示用的prompt模板，回复"选择链"获取。

3. 上期聊"理解"，这期聊"创作"，下期聊"如果AI能做一切，那我该做什么？"——建议按顺序看。

4. 你的"选择链"有多长？评论区分享你的做法👇
```

### 预设回复模板（4套）

**模板1**：针对"AI到底能不能创作"
```
recombine是重组已有模式，select是选择值得表达的。区别不在于结果好不好，而在于有没有那个"为什么"。万花筒也有好看的图案，但图案背后没有一个会死、会后悔、会珍惜的人在说"这就是我想说的"。
```

**模板2**：针对"番茄炒蛋破防了"
```
真实的故事是最硬的通货。AI能生成1000个关于失去母亲的句子，但永远不会知道：番茄炒蛋是妈妈最常做的菜。这个"为什么"，只有你知道。这就是你的创作。
```

**模板3**：针对"太实用了"
```
核心就一句话：以前你说"帮我写首诗"，现在你说"给我10个方向，我来选"。把AI当万花筒，别把它当创作者。完整方法在置顶评论。
```

**模板4**：针对"和上期的关系"
```
上期聊理解（correlate vs understand），这期聊创作（recombine vs select）。共同点：AI在用错误的动词。下期聊存在——如果AI能做一切，你该做什么？
```
