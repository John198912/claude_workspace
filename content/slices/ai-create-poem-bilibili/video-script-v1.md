# B站视频口播逐字脚本

## 脚本符号说明

| 符号 | 含义 |
|------|------|
| [0:00] | 时间码（分:秒） |
| 【画面】 | 此时出现的画面内容 |
| 【音效】 | 音效/BGM变化 |
| // | 口播停顿（约1-2秒） |
| /// | 口播长停顿（约3秒以上） |
| *斜体* | 口播中的情感/语气标注 |
| → | 画面切换/转场 |

## 第2章：万花筒——AI工作原理（3:30-7:00）

---

**[3:38]**

【画面】B-roll：真实万花筒转动特写，彩色碎片旋转。然后过渡到动画版万花筒——碎片变成文字（"月光""影子""寂静""空荡"）。

【音效】万花筒转动 + 冷电子BGM

**口播**（画外音）：

AI的创作，就像这个万花筒。

/

万花筒里面有彩色碎片。每转一次，碎片重新排列。

/

每个图案都"新"——但都"一样"。

因为从头到尾就是那些碎片。

---

**[4:10]**

【画面】三步流程图动画。第一步：输入框"写一首关于孤独的诗"→"孤独""诗""情感"三个词高亮弹出。

**口播**：

AI写诗，分三步。

/

第一步，识别关键词。

你输入"写一首关于孤独的诗"，AI识别到三个关键词：孤独、诗、情感。

---

**[4:40]**

【画面】第二步：数据库图标旋转，关联词逐条弹出——月光0.89、空荡0.87、影子0.85、寂静0.83——每条用柱状图可视化概率。

**口播**：

第二步，检索相关元素。

/

AI在它读过的所有文本里找："孤独"这个词，最常和哪些词一起出现？

/

月光、空荡、影子、寂静。它们和"孤独"的关联度都在0.85以上。

---

**[5:15]**

【画面】第三步：碎片从四面八方飞入，旋转，组合成完整诗句。

**口播**：

第三步，重组元素。

/

从10亿种可能的组合里，找到统计上最优的那一个。

"月光 + 空荡 + 房间 + 影子 + 陪伴"——概率最高。输出。

---

**[5:45]**

【画面】左右分屏。左侧万花筒转10次——每次碎片相同。右侧10首"不同"的诗逐行叠加，半透明层累积后露出共用核心词汇的热力图。

**口播**：

所以你看，AI写10首诗。

/

每次都"新"——因为组合不同。

/

但每次都"一样"——因为碎片没变。

///

---

**[6:30]**

【画面】切回创作者正脸。背景暗色，脸部由暖光照明。

**口播**：

所以当AI说"我创作了一首诗"——

/

它其实在说：*慢* "我从10亿种元素组合里，找到了统计上最优的那一个。"

/

*停顿*

这不是创作。

/

这是……

//

重组。

---

**[6:55]**

【画面】文字卡叠入："recombine（重组）"——英文高亮，下方中文字幕。

**[7:05]**

【画面】黑白纪录片B-roll：工匠用锤子和凿子敲击大理石，碎屑飞溅。快放——粗糙石块逐渐显现人形轮廓。叠加字幕：

**口播**（画外音，语速慢）：

人类的创作不一样。

/

米开朗基罗说过一句话：雕塑已经在石头里了，我只是把多余的部分去掉。

/

这就是人类的创作。不是加法。是减法。

///

他划掉了。

太直白。

太常见。

/

这就是了。

**[9:30]**

【画面】第三稿金色高亮，放大。其余文字全部消失。画面只剩这两行金色文字停留3秒。

【音效】完全静默3秒

/// （3秒静默）

---

**[9:33]**

【画面】三稿删除过程慢动作回放。每次删除时，画面出现计数器：-1、-2、-3……文字叠加：

"每一次删除，都是一次选择。"

"每一次选择，都是一次放弃。"

【音效】温暖的钢琴单音缓慢进入

**口播**（画外音，语速慢）：

注意这个过程。

/

诗人不是在添加元素——而是在删除可能性。

/

每一次删除，都是一次选择。每一次选择，都是一次放弃。

///

从"我很孤独"，到"我妈妈去世后，我学会了一个人吃饭"——这句话之所以打动你，不是因为加了多少东西，而是因为诗人知道：什么不该说。

---

**[10:30]**

【画面】动画：粗糙石头 → 雕刻刀一刀刀切掉多余部分 → 大卫像显现。最后刀停留在关键细节上。

**口播**：

这里有个有意思的点。

/

对AI来说，所有元素都是平等的。月光、影子、寂静、空荡——都是概率分布上的一个点，权重不同，但没有哪个是"多余的"。

/

但对人来说，有些东西是必须说的，有些东西是不能说的。

/

我的一个理解是：AI做不到减法，不是因为它不够聪明，而是因为——它没有"多余"这个概念。

/

而人类的创作，恰恰就是知道什么是"多余"。

///

这个区别，就是创作。

## 第4章：命名时刻+哲学支撑（11:00-15:00）

---

**[11:10]**

【画面】全屏对比图。左侧冷蓝色调，右侧暖橙色调。中间一条竖线。

左列逐行弹出：
- AI = 万花筒
- recombine（重组）
- 加法逻辑
- 不知道"为什么"

右列逐行弹出：
- 人类 = 雕刻刀
- select（选择）
- 减法逻辑
- 知道"为什么值得"

【音效】沉思弦乐，音量极低

**口播**：

现在，可以给这个区别一个名字了。

/

*慢*

AI的动词，是 recombine，重组。

人类的动词，是 select，选择。

/

AI在做加法——不断添加元素。

人类在做减法——不断删除可能性。

/

AI不知道"为什么这个"。

人类知道"为什么值得"。

///

---

**[12:30]**

【画面】棋盘动画。中间是一个棋盘（围棋/国际象棋不限，简洁几何风格）。AI（机械臂图标）站在一侧——棋盘上的所有格子亮起，展示AI知道所有走法规则。人（简笔画图标）站在另一侧——棋盘上浮现一张照片（母亲）、连接线（延伸到观众方向）。

【音效】弦乐保持，极低

**口播**：

这个区别，哲学家维特根斯坦也给过我们一个视角。

/

他说：语言的意义在于使用。

/

什么意思？语言不是用来描述世界的，而是用来做事的。

就像"我承诺"——不是在描述一个承诺，而是在做出承诺。

/

创作也是这样。

不是发现真理，而是参与一个游戏。

/

AI知道游戏规则——如何组合词语、如何押韵、如何符合语法。

/

但不知道为什么玩这个游戏——为什么要表达这个、为什么是这个时刻、为什么这种孤独值得被写出来。

/

人类知道为什么玩。所以每次玩都不一样。

---

**[14:05]**

【画面】动画：一块画布铺满密密麻麻的文字。雕刻刀从左上角开始，一刀一刀精准删除。最终只剩一句话："我妈妈去世后，我学会了一个人吃饭。"这句话放大，金色。

**口播**：

所以真正的创作是减法。

/

不是我要说什么。而是我不说什么。

/

对AI来说，所有答案都是平等的——都是概率选择。

/

但对人类来说，有些东西是必须说的，有些东西是不能说的。

/

这个"必须"和"不能"——就是创作。

/

——AI永远不会有。

## 第5章：实机演示（15:00-19:30）

---

**[15:10]**

【画面】切回创作者正脸。镜头中景。语气从沉思转为轻快。

【音效】BGM切换到轻快电子/钢琴

**口播**：

OK，到这里你可能在想：那我用AI创作还有什么意义？

/

有意义。

/

因为AI可以提供"万花筒"，但你要做"雕刻刀"。

/

具体怎么做？我给你看。

---

**[15:35]**

【画面】全屏屏幕录制。右下角画中画创作者小窗。打开ChatGPT（或类似AI工具）。

**口播**（边操作边解说）：

我们以写文章为例。

假设我是一个自媒体人，想写一篇关于失去母亲的文章。

---

**[15:50]**

【画面】屏幕录制。在ChatGPT输入框输入："我想写一篇关于失去母亲的文章。给我5个完全不同的开头方向，每个用一句话。不要写完整文章，只给开头方向。"

AI输出5个方向。光标在第5个"我妈妈去世后…"上停留。

**口播**：

第1步：让AI生成5个开头。

不要只生成1个——1个你只能接受或拒绝。5个你才需要选择。

/

你看AI给了5个方向。我选择了第5个："我妈妈去世后……"

/

为什么选这个？因为这是我最真实的经历。我妈妈确实去世了。

/

这就是第一次选择——你知道什么是真的。

---

**[16:30]**

【画面】继续操作。输入："基于第5个方向'我妈妈去世后'，给我3个不同风格的扩展版本：A.煽情版 B.克制版 C.哲理版"

AI输出3个版本。鼠标在B版本上停留，选中其中部分内容。

**口播**：

第2步：让AI扩展这个方向。要3个不同风格的版本。

/

煽情版、克制版、哲理版。

/

我选择了克制版的结构——因为我不想消费自己的悲伤。

/

但AI在克制版里加了很多"人生感悟"——那些看起来很深刻但没什么信息量的句子。

/

我删掉。全部删掉。

---

**[17:10]**

【画面】屏幕录制。在AI输出上直接划线——大段文字被删除，只剩下几行骨架。然后在空白处手动打字："第一次，我煮了她最爱的番茄炒蛋。我吃了一口，哭了。"

**口播**：

第3步：删除。AI生成的内容，我大概删了80%。

/

删完之后，我坐在电脑前，想起了一件事。

/

*声音变轻*

我妈妈最爱做番茄炒蛋。每次我不开心，她就会做。

/

我把这个细节加进去了。

/

"我吃了一口，哭了。"

/

这句话AI写不出来。不是因为它不会组合这些词——它当然会。而是因为它不知道：番茄炒蛋是什么味道。在那个时刻，那口番茄炒蛋意味着什么。

---

**[17:50]**

【画面】屏幕录制。最终文案全屏展示：
"我妈妈去世后，我学会了一个人吃饭。
第一次，我煮了她最爱的番茄炒蛋。
我吃了一口，哭了。"

文字缓慢放大。

**口播**：

*慢*

这就是最终的文案。

/

AI提供了开头方向和结构框架。但"番茄炒蛋"和"哭了"——是我的。

/

每一次选择、每一次删除、每一次加入——加起来，就是我的创作。

/

我称之为：选择链。

---

**[18:20]**

【画面】动画：链条从左到右延伸。每环一个图标：5选1 → 3选1 → 删80% → +番茄炒蛋 → 最终文案。链环逐一亮起，颜色从冷蓝渐变为暖橙。

【音效】每个链环亮起时配轻快的确认音效

**口播**（画外音）：

每个环节都是你的一次选择。所有选择串起来，就是你的选择链。

/

选择链越长，你的参与度越高，作品越"你"。

---

**[18:50]**

## 录制说明

1. **语速建议**：正常语速约180-200字/分钟。以上口播约2800字，纯口播时间约14-15分钟。其余时间留给画面切换、静默、动画展示和实机操作。

2. **实机操作排练**：第5章的屏幕录制prompt流程需提前排练5遍以上，确保流畅不卡顿。操作过程和口播要同步。

3. **情感段落**：第3章（我妈妈去世后）和第6章（回到开场）是情感高点，语速需要明显放慢，留出情感空间。

4. **录制方式**：建议分章录制，而非一次性录完全程。每一章可以单独录3-4遍，选最好的一遍。

5. **画中画位置**：屏幕录制时画中画放在右下角，不要遮挡AI输出文字的关键部分。
