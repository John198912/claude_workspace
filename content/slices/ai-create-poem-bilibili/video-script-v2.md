## 第2章：万花筒——AI工作原理（3:05-6:30）

**口播**（画外音）：

AI的创作，就像一个万花筒。

/

---

**[3:42]**

【画面】三步流程图动画。第一步：输入框→"孤独""诗""情感"高亮。

**口播**：

AI写诗，分三步。

/

第一步，识别关键词。你输入"孤独"，AI识别出：孤独、诗、情感。

---

**[4:10]**

【画面】第二步：数据库旋转，关联词弹出——月光0.89、空荡0.87、影子0.85——柱状图可视化。

**口播**：

第二步，找关联。AI在它"读过"的所有文本里搜：孤独最常和哪些词一起出现？

/

月光、空荡、影子、寂静。关联度都在0.85以上。

---

**[4:45]**

【画面】第三步：碎片从四面八方飞入，旋转组合成诗句。

**口播**：

第三步，重组。从10亿种组合里，找统计上最优的那一个。

/

"月光+空荡+房间+影子"——概率最高。输出。

---

**[5:15]**

【画面】分屏：左-万花筒转10次（相同碎片），右-10首"不同"的诗叠加，半透明层露出热力图。

每次都"新"，但每次都"一样"。

**[5:55]**

**口播**：

## 第4章：命名时刻+哲学支撑（10:30-15:00）

---

**[10:38]**

【画面】全屏对比图。左冷蓝，右暖橙。中间竖线。

左列弹出：AI=万花筒 / recombine / 加法 / 不知道"为什么"
右列弹出：人类=雕刻刀 / select / 减法 / 知道"为什么值得"

【音效】沉思弦乐，极低

**口播**：

现在，可以给这个区别一个名字了。

/

*慢*

AI的动词，是 recombine——重组。

人类的动词，是 select——选择。

/

AI在做加法。人类在做减法。

/

AI不知道"为什么这个"。人类知道"为什么值得"。

///

这张图你可以截个图——或者打1告诉我你记住了。

---

**[12:00]**

【画面】棋盘动画。AI（机械臂）在一侧，格子全亮。人（简笔画）在另一侧，棋盘浮现照片、连接线。

【音效】弦乐保持，极低

**口播**：

这个区别，维特根斯坦也给过一个视角。

/

他说：语言的意义在于使用。

/

什么意思？语言不是用来描述世界的，而是用来做事的。

/

"我承诺"——不是在描述一个承诺，而是在做出承诺。

/

"对不起"——不是在陈述一个事实，而是在道歉。

/

你想想日常。你跟朋友说"下次一起吃饭"——你真的在约定时间吗？不是。你是在表达"我们的关系还在"。

/

这就是语言游戏。每个词都在做一件事，不是描述一件事。

---

**[13:10]**

【画面】棋盘上AI侧的棋子按规则排列；人侧的棋子上浮现个人故事。

**口播**：

创作也是语言游戏。不是发现真理，而是参与一个游戏。

/

AI知道所有规则——如何组合词语、如何押韵、如何符合语法。这就像一个人知道围棋所有走法。

/

但不知道为什么要下这盘棋——为什么是这个时刻、为什么这种孤独值得被写出来。

/

人类知道为什么玩。因为我们有故事。有妈妈。有那个吃了会哭的番茄炒蛋。

///

所以每次玩都不一样。

---

**[14:05]**

【画面】动画：铺满文字的画布→雕刻刀逐刀精准删除→只剩一句话，放大，金色。

**口播**：

所以真正的创作是减法。

/

不是我要说什么。而是我不说什么。

/

对AI来说，所有答案都是平等的。对人类来说，有些东西必须说，有些东西不能说。

/

"必须"和"不能"——就是创作。

/

AI永远不会有。
