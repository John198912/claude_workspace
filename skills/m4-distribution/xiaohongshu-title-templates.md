## 使用说明

这75个标题模板来自dbskill的实战验证,每个模板都经过真实数据验证。使用时:
1. 根据内容类型选择合适的模板类别
2. 填入具体的关键词和数值
3. 保持标题在20字以内(最佳)
4. 第一行必须包含核心关键词(SEO)

## 标题优化规则

### 规则1: 关键词前置
```
❌ "我用这个方法,30天涨了1000粉"
✅ "涨粉1000+!30天实战方法"
```

### 规则2: 数字具体化
```
❌ "AI工具推荐"
✅ "5个AI工具,效率提升10倍"
```

### 规则3: 制造反差
```
❌ "AI写作教程"
✅ "AI写作 vs 人工写作,差距惊人"
```

### 规则4: 情感词汇
```
❌ "内容创作方法"
✅ "内容创作者的心酸,只有创作者懂"
```

### 规则5: 长度控制
- 最佳长度: 15-20字
- 最大长度: 25字
- 核心关键词: 前10字内

## 集成到s-4.2.2-adapt

```python
def generate_xiaohongshu_title(content, keywords):
    """
    基于内容和关键词生成小红书标题

    Args:
        content: 原始内容
        keywords: 核心关键词列表

    Returns:
        titles: 3个备选标题
        template_used: 使用的模板
    """

    # Step 1: 分析内容类型
    content_type = analyze_content_type(content)

    # Step 2: 匹配模板类别
    template_category = match_template_category(content_type)

    # Step 3: 选择3个模板
    templates = select_templates(template_category, n=3)

    # Step 4: 填充模板
    titles = []
    for template in templates:
        title = fill_template(template, keywords, content)
        titles.append(title)

    # Step 5: 优化和验证
    titles = [optimize_title(t) for t in titles]

    return titles, templates
```
