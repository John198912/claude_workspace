# 自我SDF——通过信息环境策展重塑价值观

## 核心观点

你可以通过策展自己的信息环境来重塑价值观——就像Anthropic通过宪法文档和虚构故事改变AI的角色先验。但工具的伦理边界是自反性：如果你不知道你为什么选择这个信息环境，你不审视正在发生的变化——那是洗脑。如果你在知情的情况下选择，持续反思，保留停止的权利——那是自我SDF。

## 详细说明

### 灵感来源

Anthropic的SDF（Synthetic Document Fine-Tuning）：在微调前，用宪法文档和虚构故事训练基础模型——不是教它"做什么"，而是让它浸泡在"这样的AI是存在的"叙事中。结果：角色认同改变→行为在OOD场景中自然对齐。

### 操作流程

1. **审计信息食谱**：你每天花时间在什么内容上？它们正在把你塑造成什么样的人？
2. **定义目标方向**：你想在哪个方向上调整价值观？
3. **策展信息环境**：用故事替代论证（故事让人自己推理出原则）；用浸润替代突击（每天30分钟×3个月 > 一个周末集训）

### ⚠️ 五条安全规则

1. 你必须能说清楚"我为什么选择这个信息环境"
2. 你可能会失去一些东西（旧关系、确定感、"不知道更好"的舒适感）
3. 每月审计一次："这个改变是我选择的结果，还是无意中的漂移？"
4. 不要用自我SDF逃避"现在就要做决定"的痛苦
5. 你不必永远做自己的雕塑家——有些时候漂移是可接受的

### 自我SDF vs 自我洗脑的边界

边界 = 自反性。知情选择 + 持续反思 + 保留停止权利 = 自我SDF。不知道自己在被改变 + 不审视变化过程 = 洗脑。

## 关键概念

- **SDF**：Synthetic Document Fine-Tuning，通过文档浸润改变先验
- **惯习（Bourdieu）**：通过社会环境浸润形成的隐性认知和行为倾向
- **自反性**：对自己思考过程进行审视的能力

## 实践应用

- 手机主屏放阅读app而非社交媒体（改变默认的信息环境）
- 通勤时听你想成为的那类人的播客
- 睡前读你想内化的那种故事（而非刷社交媒体）
