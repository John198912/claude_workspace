# 「卖产品赚钱」不是一件事，是 N 件事

## 知识陈述

「卖产品赚钱」不是一件事，是 N 件事

## 原始推文

> 「卖产品赚钱」不是一件事，是 N 件事\n\n第一层是有人需要这个产品\n\n第二层是有人愿意为这个产品付钱\n\n第三层是有很多人愿意为这个产品付钱\n\n第四层是我可以持续性地获取流量，源源不断地让很多人为这个产品付钱\n----\n\n以及，可能还会有\n\n第五层：从流量到品牌，从依赖持续获客到依赖客户忠诚\n\n第六层：从单一产品到多产品协同，从「满足一个需求」到「满足一系列需求」\n\
