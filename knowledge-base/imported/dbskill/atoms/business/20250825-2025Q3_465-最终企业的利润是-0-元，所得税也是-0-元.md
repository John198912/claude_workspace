# 最终企业的利润是 0 元，所得税也是 0 元

## 知识陈述

最终企业的利润是 0 元，所得税也是 0 元

## 原始推文

> 税法承认企业为了赚钱必须先花钱，但不承认个人为了生活必须先花钱\n\n假设同样是年入百万，全部花掉，手里不留一分钱\n\n企业主可以把这 100 万全部作为合理经营成本花掉，这里面可能可以包括吃饭、酒店、机票，甚至买车等等\n\n最终企业的利润是 0 元，所得税也是 0 元\n\n但是上班的情况下，可能需要交 35 万的个人所得税，实际购买力是 65 万\n\n也就是说只承认企业可以通过支出成本来
