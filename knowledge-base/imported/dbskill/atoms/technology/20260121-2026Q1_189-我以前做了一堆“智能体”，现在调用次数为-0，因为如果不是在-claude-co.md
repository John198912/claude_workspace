# 我以前做了一堆“智能体”，现在调用次数为 0，因为如果不是在 claude code 调用，而是在网页、app 的对话框...

## 知识陈述

我以前做了一堆“智能体”，现在调用次数为 0，因为如果不是在 claude code 调用，而是在网页、app 的对话框里面调用，对话数据就无法沉淀，prompt 就无法迭代，conversation 数据也浪费了
