# 于是我开始引入一个「核心概念词典」，由我来口述，claude code 执笔维护

## 知识陈述

于是我开始引入一个「核心概念词典」，由我来口述，claude code 执笔维护

## 原始推文

> 我发现我和 AI 的沟通过程中，有大量的时间浪费在，我反复给 AI 解释一个「我很清楚但 AI 不知道的概念」\n\n这导致我的用户提示词不精炼，工作效率大大降低\n\n于是我开始引入一个「核心概念词典」，由我来口述，claude code 执笔维护\n\n这样一来，我和 claude code 的沟通就会非常精炼，使用大量只要我们内部知道的「黑话」\n\n最终这个词典会越来越丰富，以至于我的表达
