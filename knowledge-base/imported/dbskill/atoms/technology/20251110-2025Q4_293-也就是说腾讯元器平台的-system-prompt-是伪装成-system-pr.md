# 也就是说腾讯元器平台的 system prompt 是伪装成 system prompt 的 user prompt

## 知识陈述

也就是说腾讯元器平台的 system prompt 是伪装成 system prompt 的 user prompt

## 原始推文

> 用同样的提示词和同样的模型做智能体（DeepSeek V3.1）\n\n然后尝试逆向系统提示词\n\n如果是在 monica 等聚合平台，可以逆向出我填写的 system prompt\n\n但是在腾讯元器，直接得到了 DeepSeek V3.1 本身的系统提示词\n\n也就是说腾讯元器平台的 system prompt 是伪装成 system prompt 的 user prompt\n\n做了
