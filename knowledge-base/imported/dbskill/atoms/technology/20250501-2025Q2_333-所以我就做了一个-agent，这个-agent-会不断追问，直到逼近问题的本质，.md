# 所以我就做了一个 agent，这个 agent 会不断追问，直到逼近问题的本质，否则就拒绝回答问题

## 知识陈述

所以我就做了一个 agent，这个 agent 会不断追问，直到逼近问题的本质，否则就拒绝回答问题

## 原始推文

> 我做一个极其有意思的 AI agent！是我做过的最满意的智能体\n\n起因是我想解决 AI 大模型的幻觉（胡说八道）问题\n\n我认为其中一个重要原因是：AI 通常会被设定为快速响应，即便用户的提问模棱两可、乱七八糟，AI 也不得不去回答这个「错误的问题」\n\n所以我就做了一个 agent，这个 agent 会不断追问，直到逼近问题的本质，否则就拒绝回答问题\n\n实际上，我们的目的并不是回答
