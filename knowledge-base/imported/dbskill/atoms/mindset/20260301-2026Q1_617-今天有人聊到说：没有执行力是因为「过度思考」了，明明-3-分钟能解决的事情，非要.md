# 今天有人聊到说：没有执行力是因为「过度思考」了，明明 3 分钟能解决的事情，非要浪费一小时去规划

## 知识陈述

今天有人聊到说：没有执行力是因为「过度思考」了，明明 3 分钟能解决的事情，非要浪费一小时去规划

## 原始推文

> 今天有人聊到说：没有执行力是因为「过度思考」了，明明 3 分钟能解决的事情，非要浪费一小时去规划\n\n我认为这个说法，是对不愿意执行的美化\n\n执行就意味着两个风险：\n1、执行了就有可能失败\n2、即便执行成功了，你可能依然还是平庸（比如工作问题解决了，在社会层面上依然不是成功人士）\n\n所以在不执行的情况下，只是去推理、去思考、去模拟，就可以收获「可能性的快感」\n\n应该是齐泽克说过一
