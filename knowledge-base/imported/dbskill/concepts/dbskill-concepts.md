## 概述

- "产品" (593次) 是绝对主导概念

- 融合了语言哲学和心理学框架

## 主题聚类分析

### 聚类1: 商业模式与定价 (最活跃)
**核心概念**: 产品、需求、定价、商业模式、变现、转化
**关键洞察**:
- 定价基于市场而非成本
- 产品必须可交易 (能发付款链接)
- 流量不等于赚钱

### 聚类2: 内容创作与平台
**核心概念**: 内容、平台、算法、流量、对标
**关键洞察**:
- 平台算法决定内容分发
- 对标优于原创 (0到1阶段)
- 不同平台有不同激励机制

### 聚类3: AI工具与应用
**核心概念**: 提示词、智能体、AI、模型
**关键洞察**:
- 提示词是AI交互的核心
- AI放大现有能力,不创造能力
- 工具服务于商业模式,而非反之

### 聚类4: 执行与方法论
**核心概念**: 执行、SOP、对标、问题消解
**关键洞察**:
- 执行100次优于完美计划
- SOP是规模化的前提
- 对标消解"我该怎么做"的问题

### 聚类5: 哲学与心理学
**核心概念**: 维特根斯坦、阿德勒、认知、齐泽克
**关键洞察**:
- 语言边界即思维边界
- 行动障碍源于心理而非能力
- 意识形态影响商业决策


### 概念组合
高频概念的组合往往揭示核心方法论:
- **产品 + 需求 + 定价** → 商业模式设计
- **对标 + 执行 + SOP** → 0到1方法论
- **内容 + 平台 + 算法** → 内容分发策略
- **提示词 + 智能体** → AI工具应用

### 概念演化
追踪概念在不同季度的频次变化:
- 2024Q4-2025Q1: "知识付费"高频
- 2025Q2-Q3: "智能体"崛起
