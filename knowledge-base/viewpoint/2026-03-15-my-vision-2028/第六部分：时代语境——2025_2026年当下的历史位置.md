## 第六部分：时代语境——2025/2026年当下的历史位置

### 7.1 我们在历史的哪个坐标上？

历史从来不会宣布自己的转折点。

1789年的法国人不知道自己正在经历
"封建制度的终结"；
1914年的欧洲人不知道自己正在走向
"旧世界秩序的崩溃"。
转折点只有在事后才能被清晰命名。

但有一种方法可以在当下感知历史位置：
**观察症状的密度和速度。**

当一个文明的圣状开始被根本性动摇，
症状会以加速度的方式在各个领域同时涌现——
经济、政治、文化、心理，
没有一个领域能够幸免。

2025-2026年，我们正在目睹这种症状的全面爆发。

---

### 7.2 经济症状：数据说话

#### 劳动市场的结构性断裂

2025-2026年的劳动市场数据，
正在呈现出一种前所未有的分裂结构。

Anthropic的研究显示：
在ChatGPT出现之后的时代，
**AI高暴露职业的求职成功率
平均下降了14%**——
这不是周期性波动，而是结构性断层。
更令人警醒的是，
对22至25岁年轻工作者的研究发现，
AI暴露职业中的就业率
**下降了16%**，
研究者将其比作
"白领工作者的大衰退"。$CITE_3

高盛的估算更为直接：
**6%至7%的美国工人
可能因AI采用而失去工作**，
而这仅仅是当前阶段的估算，
尚未计入AGI突破后的影响。$CITE_2

与此同时，美国劳动市场出现了
一个极为反常的现象：
**2025-2026年，除医疗行业外，
美国月度新增就业岗位几乎为零**——
经济在增长，GDP在上升，
但就业岗位没有随之增加。
这正是"幽灵GDP"现象的现实显现：
财富在被创造，但创造财富不再需要人。$CITE_3

#### 算力投资与财富集中的加速

与就业停滞形成鲜明对比的，
是算力投资的爆炸性增长。

2026年，AI相关资本支出
预计将超过**5000亿美元**，
占GDP的比例接近1.5%，
达到历史上其他技术繁荣时期的峰值水平。$CITE_2

圣路易斯联储的数据显示：
仅在2025年前三季度，
AI相关类别就为实际GDP增长
**贡献了0.97个百分点**。$CITE_3

但这些财富的流向高度集中：
分析师Azeem Azhar指出，
AI公司2025年预计创造**600亿美元收入**，
但投入了**4000亿美元支出**——
这意味着巨额财富正在从
整个经济体系流向极少数算力所有者，
形成历史上规模最大的单向财富转移之一。$CITE_4

```
2025-2026年经济症状全景：

就业端：
├─ AI高暴露职业求职成功率下降14%
├─ 年轻白领就业率下降16%
├─ 非医疗行业月度新增就业近乎为零
└─ 白领失业潮开始显现

财富端：
├─ AI资本支出超5000亿美元/年
├─ AI为GDP贡献近1个百分点
├─ 但财富高度集中于算力所有者
└─ 600亿收入vs4000亿支出的财富虹吸结构

结构性特征：
GDP增长与就业增长的历史性脱钩
→ 这不是衰退，而是新型财富集中模式的确立
```

### 7.3 地缘政治症状：竞争囚徒困境的全面激活

如果说经济症状是AI时代圣状危机的内部显现，
那么地缘政治症状则是它的外部放大器。

#### 美中AI军备竞赛的结构性锁定

2026年，美中之间的AI竞争
已经从"技术竞争"演变为
**全面的战略对抗**。

大西洋理事会的分析识别出
2026年AI地缘政治的八个关键走向，
其中最核心的两个是：
**美国推动AI技术出口以对抗中国**，
以及**美中AI竞赛在DeepSeek冲击后进一步激化**。$CITE_2

中国的回应同样清晰：
2026年全国人大批准的五年计划，
将**AI驱动的技术发展和科学研究
列为最高国家优先级**，
机器人和AI工厂成为核心战略目标。$CITE_4

高盛全球研究所的分析指出：
尽管贸易紧张关系短暂缓和，
但**技术竞赛正在加速**——
两国都以"对方在加速"为由
为自己的加速提供合法性，
形成了一个经典的**竞争囚徒困境**：
每个参与者的理性选择（加速）
导致了对所有参与者都更糟糕的集体结果。$CITE_1

#### 竞争囚徒困境的意识形态效果

这个地缘政治结构，
在意识形态层面产生了极为强大的效果：

```
竞争囚徒困境的意识形态锁定：

对美国公众：
"中国在加速，我们不能落后"
→ 国家安全叙事压制了
  对AI发展方向的内部批评

对中国公众：
"美国在封锁，我们必须自强"
→ 民族主义叙事将AI发展
  与国家尊严绑定

对全球其他国家：
"必须选边站，必须跟上"
→ 没有任何主要国家
  有政治空间踩刹车

最终效果：
全球范围内，
"是否应该以这种方式发展AI"
这个根本性问题
被"如何在AI竞争中不落后"
这个操作性问题完全替代
```

### 7.4 政治症状：合法性危机的多点爆发

经济结构的断裂和地缘政治的锁定，
正在以可预测的方式转化为政治危机。

#### 传统政治框架的全面失效

2025-2026年，
几乎所有主要民主国家都在经历
同一种政治症状：

**传统的左右翼政治框架
无法有效回应AI时代的核心矛盾，
导致选民基础的加速侵蚀
和极端主义的同步崛起。**

这不是偶然的政治波动，
而是圣状危机在政治层面的系统性显现：

```
政治症状的结构性来源：

传统右翼的困境：
支持市场和技术发展（资本逻辑）
vs. 选民要求保护工作岗位（劳工逻辑）
→ 无法同时兑现两个承诺

传统左翼的困境：
依赖工会和劳工运动（组织基础）
vs. 工会力量随AI替代而系统性衰减
→ 政治主体基础被釜底抽薪

民粹主义的困境：
用"夺回工作"动员选民（执政承诺）
vs. 技术替代趋势不可逆（现实约束）
→ 执政后必然面临承诺破产

自由主义精英的困境：
相信"技术进步+再分配=所有人受益"
vs. 数据显示财富集中在加速
→ 叙事与现实的裂缝越来越大
```

#### 监管的系统性滞后

与政治框架失效并行的，
是监管能力的系统性滞后。

大西洋理事会的分析指出，
2026年**AI治理正在走向全球化**，
但这个"走向"本身就说明：
有效的全球AI治理框架
目前仍然不存在。$CITE_2

更深层的问题是：
监管的速度从结构上就无法匹配技术发展的速度。
技术突破以月为单位，
立法程序以年为单位，
国际条约以十年为单位——
这个时间尺度的错配，
不是执行力问题，而是制度设计的根本性局限。

### 7.5 文化与心理症状：意义危机的大众化

经济和政治症状是可以被数据量化的，
但文化和心理症状同样真实，
且在某种意义上更为根本——
因为它们直接触及圣状的核心。

#### "有用性焦虑"的全面蔓延

2025-2026年，
一种新型的心理症状正在大规模蔓延，
我们可以称之为**"有用性焦虑"**：

```
有用性焦虑的现象学描述：

不是对失业的恐惧（那是经济焦虑），
而是对"我还有没有存在价值"的深层疑问。

典型表现：
├─ 知识工作者在使用AI工具时
│   感到自己的专业技能被"空心化"
├─ 创作者在AI生成内容面前
│   质疑自己创作行为的意义
├─ 学生在AI可以完成所有作业的情况下
│   不知道"学习"的目的是什么
└─ 管理者在AI可以优化所有决策的情况下
   不确定自己的判断是否还有价值

这种焦虑的深层结构：
它不是"我会失业吗"（可以通过找新工作解决），
而是"如果AI能做所有有用的事，
我做什么才算有意义"
（这个问题没有简单的答案）
```

这种焦虑的大众化，
是劳动圣状被动摇的最直接的心理显现。
它标志着：
AI对劳动的冲击已经从经济层面
渗透到了文明意义层面。

#### 注意力经济的终极形态

与有用性焦虑并行的，
是注意力经济在AI时代的终极进化：

AI不再只是捕获注意力的工具，
它开始成为**生成意义的工具**——
人们越来越多地通过与AI的对话
来处理自己的情感、做出决策、
寻找人生方向。

这个趋势的深层含义是：
**意义的生产，正在从人类集体实践
转移到算法系统。**

而意义的生产者，
就是新时代最深层的权力所有者。

### 7.6 四个关键摩擦点：历史分叉的可能位置

在所有这些症状之中，
有四个**关键摩擦点**——
在这些点上，
现有系统的内在矛盾最为尖锐，
历史分叉最有可能在这里发生。

---

**摩擦点一：白领失业潮与中产阶级政治**

```
摩擦点描述：
AI对白领工作的替代，
正在冲击现代民主政治的核心支柱——
中产阶级。

历史意义：
工业革命的技术替代主要冲击蓝领工人，
中产阶级是稳定器。
AI时代的技术替代直接冲击中产阶级，
稳定器本身开始动摇。

关键数据：
Anthropic研究显示，
AI高暴露职业（多为白领）
求职成功率下降14%，
年轻白领就业率下降16%。$CITE_1

可能的分叉：
├─ 路径A：中产阶级政治动员，
│   推动实质性的AI监管和再分配改革
└─ 路径B：中产阶级向民粹主义转向，
    将经济焦虑转化为文化战争
```

---

**摩擦点二：AI投资泡沫与现实收益的落差**

```
摩擦点描述：
AI投资规模与实际经济收益之间
存在巨大落差，
泡沫破裂的风险正在积累。

关键数据：
2025年AI公司：
600亿美元收入 vs 4000亿美元支出
投入产出比约为1:6.7$CITE_4

AI资本支出占GDP比例
已接近历史技术繁荣峰值，
但就业和生产率的提升
尚未出现与之匹配的增长。$CITE_2

可能的分叉：
├─ 路径A：AI生产率红利在2026-2028年
│   大规模释放，验证投资逻辑，
│   但财富集中问题进一步加剧
└─ 路径B：泡沫破裂，
    AI投资热潮急剧降温，
    但圣状危机并不因此消失，
    只是以不同的形式重新出现
```

**摩擦点三：地缘政治竞争与全球治理真空**

```
摩擦点描述：
美中AI竞争的加速，
正在系统性地摧毁建立全球AI治理框架
所需要的合作基础。

现实状况：
美国推动AI技术出口管制，
将算力资源武器化；
中国加速自主AI生态建设，

治理真空的危险：
在没有全球治理框架的情况下，
AI发展的速度和方向
完全由地缘政治竞争逻辑决定，
人类福祉的考量被系统性地边缘化。

可能的分叉：
├─ 路径A：某种形式的"AI不扩散条约"
│   在重大风险事件后被迫建立
└─ 路径B：竞争逻辑持续主导，
    全球治理真空长期存在，
    风险以不可预测的方式积累
```

**摩擦点四：意义危机与新文化运动的萌芽**

```
摩擦点描述：
有用性焦虑的大众化，
正在催生对新意义框架的强烈需求。
这种需求是历史上所有重大文化转型的前提。

当前信号：
├─ "慢生活"、"去数字化"运动的兴起
├─ 手工艺、在地社区、身体劳动的文化复兴
├─ 对"AI无法替代的人类体验"的重新发现
└─ 哲学、意义、存在问题的大众化讨论

深层意义：
这些文化信号，
是人类在圣状被动摇之后
本能地寻找新意义锚点的集体行为。
它们是混乱的、碎片化的，
但它们指向了一个真实的历史需求：
一套能够容纳AI时代现实的
新文明意义框架。

可能的分叉：
├─ 路径A：这些文化萌芽
│   被商业逻辑收编（"慢生活"变成消费品牌），
│   意义危机被表面缝合
└─ 路径B：文化萌芽发展成
    真正的新意义框架，
    为文明层面的创造性转化
    提供思想资源
```

### 7.7 历史位置的精确定位

综合以上所有症状和摩擦点，
我们可以给出一个尽可能精确的历史位置判断：

```
2025-2026年的历史坐标：

我们处于：
┌─────────────────────────────────────────┐
│  圣状危机的"症状爆发期"                  │
│                                         │
│  特征：                                  │
│  ├─ 经济、政治、文化症状同时全面涌现      │
│  ├─ 旧意义框架的合法性快速侵蚀           │
│  ├─ 新意义框架尚未成形                   │
│  ├─ 各主要政治力量均无有效回应           │
│  └─ 历史分叉的条件正在形成，             │
│     但分叉尚未发生                       │
│                                         │
│  类比历史时刻：                           │
│  不是1789年（革命已经爆发），             │
│  不是1750年（矛盾尚未显现），             │
│  而是1780年代——                          │
│  旧制度的合法性已经开始崩溃，            │
│  但新秩序的形态尚不清晰，               │
│  历史的走向仍然开放                      │
└─────────────────────────────────────────┘
```

这个历史位置有一个关键含义：

**我们仍然处于历史分叉之前。**

这意味着：
当下的选择——
个人的、集体的、政治的、文化的——
仍然对历史走向有真实的影响力。

这不是乐观主义，而是历史分析的结论。

在圣状危机的症状爆发期，
系统的走向尚未被锁定。
旧的意义框架正在松动，
新的框架尚未固化——
这个"之间"的时刻，
恰恰是人类行动最有可能产生历史影响的时刻。

### 7.8 当下时刻的道德含义

本章以一个不常见的方式结束——
不是分析，而是一个道德命题。

历史分析通常以"客观中立"为旗帜，
回避价值判断。
但在一个圣状危机的症状爆发期，
"客观中立"本身就是一种政治立场——
它是对现有权力结构的默许。

因此，我们明确地提出：

```
当下时刻的道德含义：

如果我们的分析是正确的——
如果我们确实处于文明圣状的危机时刻，
如果历史分叉尚未发生，
如果当下的选择仍然有历史意义——

那么，对这个时刻保持清醒的认知，
对正在发生的事情进行诚实的分析，
对意识形态叙事保持批判性的距离，

不是学术奢侈品，
而是这个时代最基本的
知识分子责任。

把问题说清楚，
是变革的认知前提。
而认知前提，
是一切其他行动的基础。
```

这也是本文存在的理由。
